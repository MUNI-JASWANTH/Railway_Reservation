
# Release v1.0.0

**Railway Booking System** - v1.0.0

## Highlights
- Initial stable release
- Secure user registration and login (BCrypt)
- Train search, booking, cancellation, and seat management
- JSON-based local persistence and simple CLI UI

## Installation
1. `mvn clean install`
2. `mvn exec:java -Dexec.mainClass="com.irctc.App"`

## Files included
- Source code under `src/main/java/com/irctc`
- JSON data under `src/main/java/com/irctc/localdb`
- Documentation under `/docs`
